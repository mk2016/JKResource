任务状态:

- 每页显示50条任务数据

- 页面跳转控件删除

 

 

 

①筛选规则

- 新增 排序方式筛选：默认按发布时间降序排，下拉列表有：发布时间升序、发布时间降序、任务权重升序、任务权重降序四种。

 

- 按任务标题筛选，支持模糊匹配

 

- 按任务分类筛选：分为 全部、 推广、调研、互动、其他

原任务类型字段更名任务分类

 

- 按任务状态筛选：当状态选为已结束时，可继续选择是“到期结束”还是“全部完成”，还是“已下架”。

 

- 按用户姓名筛选更名为“雇主姓名”筛选

 

- 按发布时间筛选；筛选出特定时间段内发布的任务

 

- 新增 截至时间筛选：筛选出特定时间段内截至的任务

 

-新增 注册号码筛选：按雇主的注册号码筛选出该号码发布的任务。

-（新增）通过率筛选：通过率升序和通过降序筛选

 

 

 

②表单字段

- 发布时间：要分两行显示，缩减列宽

 

- 任务标题

- （新增）任务ID字段，每个任务有一个唯一的id。

- （新增）主任务ID字段，该任务关联的主任务的id；无关联任务则为-

- （删除）说明图片：点击可查看图片

 

- 任务类型：分为 推广、调研、互动、其他

 

- 用户姓名更名为“雇主姓名”：发布任务的雇主姓名，姓名底下显示该用户发布过的宅任务条数，点击跳转到【用户详情-宅任务列表】

 

- 任务数量：任务总共有几次

 

- 剩余量：该任务还剩下多少次可以被领取，当兼客领了任务两小时不执行则该次数会返回到剩余次数。

 

-（新增）已领取量：统计该任务历史被领取过的次数 ，不去重。（注意：已领取量=已提交量+已超时量）

 

- 正在执行量更名为“待提交量”：兼客领取了任务还没提交审核材料的这两小时之内的状态

 

-（新增）已超时量：统计该任务历史累计超时未提交任务材料的数量，反复超时未提交任务时不去重。

 

-（新增）已提取量：统计该任务当前已经成功提交材料的任务数量

 

- 待处理量更名为“待验收量”：处于兼客执行了任务且提交了审核材料当客服或雇主还未给出审核结果这一状态的数量。点击跳转到该任务的【任务材料列表】并筛选出“待审核”状态的提交材料。

 

- 已通过量：兼客执行任务通过审核的次数，点击跳转到该任务的【任务材料列表】并筛选出“已通过”状态的提交材料。

 

- 未通过量：兼客执行任务但未通过审核的次数。点击跳转到该任务的【任务材料列表】并筛选出“未通过”状态的提交材料。

 

- （新增）预付款：雇主支付的预付款。

 

- （新增）已支付工资：支付给审核通过的兼客的工资总额，

 

- （新增）宅任务服务费：支付给兼客平台的宅任务服务费总额

 

-（新增）分发奖励：显示该任务在审核时设置的分发奖励（审核任务页面表单文案也写成 分发奖励）

 

-（新增）已分发量：统计该任务通过分发方式，最终完成任务的用户人数（完成任务定义：任务状态处于“任务完成”状态）

 

- （新增）通过率：任务当前通过率，通过率=审核通过数量/审核的总数量

 

- 截止日期：任务截止日期

 

- 任务状态： 本页呈现：已发布、已结束（到期结束）、已结束（审核不通过）、已结束（已下架）这些状态的任务

 

 

 

-新增 任务权重字段，可调整任务权重，功能、规则与岗位列表中的一样。调整之后影响任务列表中的顺序。

 

操作：

点击编辑跳转到【编辑宅任务】页面。

点击查看跳转到【宅任务详情】页面

新增下架操作；

新增 隐藏 操作：点击后弹窗二次确认，将任务在前端宅任务列表中隐藏，但任务是有效的。

新增发短信操作，可给雇主发送短息。

 

 

 

- 新增 搜索结果条数显示。

- 新增导出功能，导出样式与列表样式一致，导出文件名为：宅任务列表-YYY.csv （YYY为导出时间，格式为：20160423） 

-导出的材料中，新增【不通过原因】字段
