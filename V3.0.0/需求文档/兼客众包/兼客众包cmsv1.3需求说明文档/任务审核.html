审核

 

页面标题由“宅任务审核”变更为“任务审核”；

菜单中新增 待审核数量气泡提示。

 

①筛选规则

 

- 按任务标题筛选，支持模糊匹配

 

- 按任务类型筛选：分为 全部、 推广、调研、互动、其他

 

- 按任务单价筛选：筛选出特定单价的任务

 

- 次数限制筛选：分为“全部”、“每人1次”和“不限”

 

- 按用户姓名筛选

 

- 按注册号码筛选

 

- 按发布时间筛选；筛选出特定时间段内发布的任务

 

②表单字段

- 发布时间

 

- 任务标题

 

- 任务链接：连接到该宅任务的任务链接页面，查看该链接的可靠性

 

 

- 任务类型：分为 推广、调研、互动、其他

 

- 用户姓名：发布任务的雇主姓名，姓名底下显示该用户发布过的宅任务条数，点击跳转到【用户详情-宅任务列表】

 

- 注册号码：雇主的注册号码

 

- 任务单价：该任务的单价

 

- 任务数量：任务总共有几次

 

  -（新增）完成率限制：运营人员可设置完成率限制大小，限制未达到该完成率的兼客，不可以领取任务。

 

- 次数限制：分为”每人1次“和”不限“

 

- 截止日期：任务截止日期

 

 

 

操作：

-审核 ：点击审核跳转到【审核宅任务】页面。

-通过：点击弹窗确认通过审核,

          发布的宅任务通过审核给雇主发送短信：“恭喜！您发布的任务【任务名称】已通过审核，请及时到兼客兼职电脑版宅任务管理中审核兼客提交的材料”；

 

-不通过：点击弹窗选择/输入不通过理由,

              发布的宅任务未通过审核给雇主发送短信：“抱歉！【客服后台填写的原因】，如有疑问请联系客服0591-63170615”；

 

审核不通过，用户支付的预付款要解冻退还到用户的钱袋子可用余额中。
